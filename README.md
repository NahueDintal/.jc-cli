## Java Commander Cli

Este es un programa tipo cli, generando los archivos necesarios para una compilación rápida en java 25.

El problema, me encontré con herramientas para Java en Linux como Maven o Gradle, es que generan proyectos con una versión anterior en caso de Maven o con formas más complejas como Gradle.

Mi perspectiva es compararlo con Dotnet que si posee dichas herramientas, por ende comence con unos script's en bash para la construcción de dichos archivos y resulta que la forma más concreta de hacerlo era construirlo con el mismo Java!

El proyecto está muy verde por el momento, y recurrí a herramientas externas como la IA para hacer parte del código, pero es cuestión de seguir haciendolo más grande y complejo.
